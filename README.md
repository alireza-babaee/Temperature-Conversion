# Temperature Conversion Tool (°C ⇄ °F)

A simple, fast, and elegant C++ console application that converts temperature between **Celsius** and **Fahrenheit**.

Perfect for beginners learning C++, or anyone who needs a lightweight temperature converter without dependencies.

### Features
- Supports both **Celsius → Fahrenheit** and **Fahrenheit → Celsius**
- Case-insensitive input (`C`, `c`, `F`, `f`)
- Clean and user-friendly interface with borders
- Proper UTF-8 support for the degree symbol (°)
- Single-file, zero dependencies — just pure C++

text### How to Use

   git clone https://github.com/alireza-babaee/Temperature-Conversion.git

Formula Used:

°C → °F: temp × 1.8 + 32

°F → °C: (temp − 32) / 1.8
